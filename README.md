# PizzaNova - Serie III

Proyecto web en React + Vite con los 5 entregables solicitados:

- 5 historias de usuario completas
- 4 historias con criterios de aceptacion
- Prototipo con 6 pantallas
- Matriz de trazabilidad
- Descripcion breve de la propuesta

## Ejecutar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Despliegue en Vercel

Vercel detecta Vite automaticamente. Solo debes importar el repositorio y usar:

- Build Command: `npm run build`
- Output Directory: `dist`
